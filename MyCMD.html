Примечание:
  • Что-бы ввести команду нажмите на клавиатуре клавишу Enter
  • Кнопка Clear служит для очищения строки для ввода
  • Любая команда начинается с "!"
  • Перед каждой командой рабочая область очищается
    • Список команд:
  • !EnglishVersion : English version installation
  • !getDate : __________Получить текущую дату__________
  • !cls : __________Очистить командную строку__________
  • !info : __________Показать основную информацию__________
  • !Vk : __________Перейти на сайт Vk.com__________
  • !Stack : __________Перейти на сайт StackOverFlow.com__________
  • !MusicVk : __________Послушать музыку ВК__________
