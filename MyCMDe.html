Note
  • To enter a command, press Enter on your keyboard
  • The Clear button clears the input line.
  • Any team begins with "!"
  • Before each command, the workspace is cleared.
    • List of commands:
  • !RussianVersion : English version installation
  • !getDate : __________Get current date__________
  • !cls : __________Clear command line__________
  • !info : __________Show basic information__________
  • !Vk : __________Go to Vk.com__________
  • !Stack : __________Go to StackOverFlow.com__________
  • !MusicVk : __________Listen music on Vk__________
